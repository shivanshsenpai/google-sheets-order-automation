# Google Sheets Order Automation

Automates:
- Tracking link generation
- Courier detection (17Track FC codes)
- WhatsApp messaging
- COD & RTO workflows
- Feedback automation
- Call tracking system

## Features
- Fully dynamic mapping
- No hardcoded IDs
- WhatsApp integration
- Smart delivery estimation

## Setup
1. Create sheets:
   - Incentive_report
   - MainData
   - CourierCodes

2. Paste Apps Script code
3. Run updateTrackingDetails()
